One-Time Setup Required

To store suggestions, you need to set up a free Supabase database:

  1. Go to supabase.com and create a free account
  2. Create a new project (any name, e.g., "tadweer-suggestions")
  3. Go to Table Editor → Create a new table named suggestions
  4. Add these columns:
    • id - uuid (default, primary key)
    • tracking_id - text
    • name - text
    • email - text
    • category - text
    • suggestion - text
    • status - text (default: 'new')
    • created_at - timestamptz (default: now())
    • page_url - text
  5. Go to Settings → API and copy your Project URL and anon/public key
  6. Go to Authentication → Policies → Enable RLS on the table, then add policies:
    • INSERT: Allow anonymous (for public submissions)
    • SELECT/UPDATE: Allow authenticated only (for admin)
